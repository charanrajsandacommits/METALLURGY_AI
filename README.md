# Metallurgy AI: Environmental Sustainability Prediction 🌍⚒️

An AI-powered Life Cycle Assessment (LCA) dashboard designed to predict and visualize the environmental impact of metallurgical mining sites. This project automates environmental compliance and helps industries transition toward circular economy models.

## 🚀 Features
- **Real-Time Sustainability Audit:** Dynamic analysis of metal types, production scales, and resource purity.
- **AI-Driven LCA Engine:** Calculates carbon footprint, water usage, and ecological risk scores.
- **Geospatial Impact Mapping:** Integrated Google Maps API to visualize site locations and environmental impact zones.
- **Mobile-Responsive UI:** Optimized for both Desktop and Mobile (APK) usage.
- **Mitigation Strategies:** Automated suggestions for reducing environmental damage based on site data.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3 (Flexbox/Responsive), JavaScript
- **APIs:** Google Maps JS API, Google Places API
- **Data:** Pandas (for master mining datasets)
- **Deployment:** Render (Web), WebToNative (Android APK)

## 📁 Project Structure
- `app.py`: Main Flask application and API routes.
- `services/`: Core logic including the `LCAEngine` and `MapsService`.
- `templates/`: HTML dashboard files.
- `static/`: CSS styling and JavaScript interactivity.
- `master_mining_data.csv`: Local dataset for metallurgical site tracking.
